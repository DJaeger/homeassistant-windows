# Home

Welcome to the **Home Assistant Windows Companion** documentation.

This application connects your Windows PC to your Home Assistant instance, providing sensors and remote control capabilities.

## Getting Started
- See the [Installation Guide](installation.md) to get the app running.
- Explore the [Features](features.md) available.
- Check [Troubleshooting](troubleshooting.md) if you encounter issues.

## About
Built with .NET 10 and WinUI 3, this app follows the standards set by the official mobile apps while providing a native Windows experience.
